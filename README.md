# 🦉 Seelaz Games

Interactive kiosk games for [Seelaz](https://seelaz.com) — play and win a coffee discount.

**▶ Play:** [elhalafawy.github.io/seelaz-games](https://elhalafawy.github.io/seelaz-games/)

---

## Games

| | Game | Goal |
|---|---|---|
| 🃏 | **Owl Memory** | Match all card pairs before time runs out |
| 🌀 | **Owl Maze** | Guide the owl to the coffee through the maze |
| 🦉 | **Pac-Owl** | Eat all pellets and dodge the ghosts |

Three difficulty levels: Easy · Medium · Hard

Win 3 stars on Medium or Hard to unlock a discount code.
